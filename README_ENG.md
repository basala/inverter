Inverter [![Built with Grunt](https://cdn.gruntjs.com/builtwith.png)](http://gruntjs.com/) [![license](http://img.shields.io/npm/l/express.svg)](https://github.com/yanhaijing/inverter/MIT-LICENSE)
========

Inverter is a leisure puzzle game, suitable for everyone. By clicking a square, the color of square block will change. Victory is achieved when all squares are turned to blue. Get your brains ready for this challenge, Inverter is very difficult.
